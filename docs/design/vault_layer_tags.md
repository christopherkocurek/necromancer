# Vault Layer Tags - The Necromancer
## VAULT-002: Layer Appropriateness Analysis

---

## Layer Assignment Criteria

Each vault is evaluated against:
1. **Depth Range** - Does the vault's min_depth match the layer?
2. **Thematic Fit** - Do monsters/terrain match layer theme?
3. **Difficulty Curve** - Is the challenge appropriate?
4. **Atmosphere** - Does the vault feel like it belongs?
