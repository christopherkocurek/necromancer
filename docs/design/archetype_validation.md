# Archetype Validation - The Necromancer
## SYN-002: Can You Build Bilbo? Faramir? Aragorn? Gandalf?

---

## Overview

This document mentally playtests four character archetypes against the Alpha 0.4.0 changes to validate that diverse stealth-capable builds are viable.

---

## ARCHETYPE 1: BILBO (Pure Stealth)

### Character Concept
- **Fantasy**: Hobbit burglar, avoid all combat, sneak to objectives
- **Primary Skills**: Stealth, Perception, Evasion
- **Playstyle**: Never fight if avoidable, find alternate routes, grab lore

### Build Path
```
Starting Skills:
- Stealth 4
- Evasion 3
- Perception 3
- Will 2

Key Abilities:
- Vanish (Stealth 6)
- Fade (Stealth 8)
- Assassination (Stealth 10)
- Exchange Places (Evasion 6)
```

### Layer-by-Layer Playtest

#### Layer 1: Forest Breach
**Viability: Excellent**
- Sleeping spiders (sleepiness 15-30) can be throat-slit
- Clear alarm creatures (crebain, squirrels) with ranged/thrown
- Use vault secret doors to bypass web clusters
- Find Ancient Glyphs, Erebor Relics for XP
- **Expected XP**: ~3,500 (descent + discovery + strategic kills)

#### Layer 2: Orc Warrens
**Viability: Very Good**
- Sleepy orcs (sleepiness 20) enable throat-slit chains
- Archers more perceptive (4) - prioritize or avoid
- Use Fade to slip past orc groups
- Barracks vaults have sleeping quarters
- **Expected XP**: ~4,500 (more discovery, some encounter)

#### Layer 3: Torture Halls
**Viability: Good**
- Ghouls have NO_SLEEP - must avoid entirely
- Dark Sorcerers can be slept and killed
- Easterlings sleepy (20) - good targets
- Start finding Thráin's Memories
- **Expected XP**: ~4,000 (avoiding combat costs kill XP)

#### Layer 4: Necropolis
**Viability: Moderate**
- All basic undead have NO_SLEEP
- Must rely on Fade, Vanish, Exchange Places
- Necromancers are priority targets (can sleep)
- High stealth needed to avoid detection
- **Expected XP**: ~3,500 (less killing, more running)

#### Layer 5: Wraith Domain
**Viability: Challenging**
- Almost universal NO_SLEEP
- Fear effects test Will investment
- Vampires have high sleepiness (20) - target these
- May need some direct combat
- **Expected XP**: ~3,000 (tough to kill anything)

#### Layer 6: Inner Sanctum
**Viability: Difficult**
- Elite enemies, most NO_SLEEP
- Black Númenóreans can be slept
- Must pick targets very carefully
- Fade + Vanish chain crucial
- **Expected XP**: ~2,500 (survival priority)

#### Layer 7: Pits of Despair
**Viability: Good**
- Extraction gameplay favors stealth
- Thráin interaction (bonus if all Memories collected)
- Can potentially avoid Sauron entirely
- Escape win is valid victory
- **Expected XP**: ~2,000 (minimal combat)

### Total Expected XP: ~23,000

### Skill Points Affordable: ~21

### Build Assessment
**Verdict: VIABLE but demanding**

Bilbo build works well in Layers 1-3, struggles in 4-6, recovers in 7. The XP rebalance ensures stealth player isn't left behind. Key challenges:
- Will investment needed for Layer 5 fear
- Some combat skill needed for emergency
- Patience required for deep undead layers

### Recommendations
- Consider +1 Perception for secret doors
- Carry escape consumables (potions of speed)
- Accept some combat in Layer 4-5

## ARCHETYPE 2: FARAMIR (Stealth Archer)

### Character Concept
- **Fantasy**: Ranger of Ithilien, bow kills from hiding, hit and fade
- **Primary Skills**: Stealth, Archery, Perception
- **Playstyle**: Ambush with arrows, retreat if detected, reposition

### Build Path
```
Starting Skills:
- Stealth 3
- Archery 4
- Perception 3
- Evasion 2

Key Abilities:
- Vanish (Stealth 6)
- Precision (Archery 6)
- Rapid Fire (Archery 8)
- Fade (Stealth 8)
```

### Layer-by-Layer Playtest

#### Layer 1: Forest Breach
**Viability: Excellent**
- Snipe alarm creatures from stealth
- Pick off sleeping spiders with arrows
- Retreat and reposition as needed
- Collect Tower Objects while exploring
- **Expected XP**: ~4,000 (encounter + kill + discovery)

#### Layer 2: Orc Warrens
**Viability: Excellent**
- Snipe orc archers (priority - high perception)
- Sleepy soldiers can be throat-slit or arrowed
- Multiple vault entries for repositioning
- Kite wargs with arrows + terrain
- **Expected XP**: ~5,000 (good kill + discovery)

#### Layer 3: Torture Halls
**Viability: Very Good**
- Snipe sorcerers before they cast
- Arrow ghouls from distance (can't throat-slit)
- Easterlings good arrow targets
- Use Fade to escape bad positions
- **Expected XP**: ~4,500 (combat effective)

#### Layer 4: Necropolis
**Viability: Good**
- Arrows effective against skeletons
- Snipe necromancers as priority
- Wights dangerous - keep distance
- Ammunition may be concern
- **Expected XP**: ~4,000 (combat viable)

#### Layer 5: Wraith Domain
**Viability: Moderate**
- Incorporeal enemies resist arrows?
- Need special arrows or melee backup
- Vampires good targets (can sleep, not incorporeal)
- Will needed for fear
- **Expected XP**: ~3,500 (arrows less effective)

#### Layer 6: Inner Sanctum
**Viability: Moderate**
- Elite HP pools strain ammunition
- Olog-hai too tanky for arrows alone
- Black Númenórean mages priority snipes
- May need melee finisher ability
- **Expected XP**: ~3,000 (ammunition limits)

#### Layer 7: Pits of Despair
**Viability: Good**
- Extraction favors hit-and-run
- Can kite final threats with arrows
- Support Sauron fight from distance
- **Expected XP**: ~2,500

### Total Expected XP: ~26,500

### Skill Points Affordable: ~22

### Build Assessment
**Verdict: STRONG and versatile**

Faramir build is one of the strongest for stealth-first gameplay. Ranged combat from hiding provides damage without engagement risk. Can handle undead layers better than pure stealth.

### Recommendations
- Invest in arrow quality (mithril arrows)
- Consider Throwing for backup damage
- Melee weapon for emergencies

## ARCHETYPE 3: ARAGORN (Stealth Melee)

### Character Concept
- **Fantasy**: Ranger, track and ambush, melee when needed
- **Primary Skills**: Stealth, Melee, Perception, Evasion
- **Playstyle**: Stalk targets, attack from ambush, disengage if overwhelmed

### Build Path
```
Starting Skills:
- Stealth 3
- Melee 4
- Evasion 3
- Perception 2

Key Abilities:
- Assassination (Stealth 10) - key ability
- Follow-Through (Melee 6)
- Riposte (Melee 8)
- Vanish (Stealth 6)
```

### Layer-by-Layer Playtest

#### Layer 1: Forest Breach
**Viability: Excellent**
- Ambush spiders for full damage
- Clear alarm creatures with melee strikes
- Use terrain from vaults
- Assassination learning target
- **Expected XP**: ~4,500 (good combat + discovery)

#### Layer 2: Orc Warrens
**Viability: Excellent**
- Throat-slit sleeping orcs (sleepiness 20)
- Engage orc groups with Riposte
- Follow-Through chains on multiple orcs
- Use vault rooms for 1v1 engagements
- **Expected XP**: ~5,500 (strong kill XP early)

#### Layer 3: Torture Halls
**Viability: Very Good**
- Melee effective against all targets
- Sorcerers die fast to Assassination
- Ghouls manageable in melee
- May take more damage than ranged builds
- **Expected XP**: ~5,000 (combat heavy)

#### Layer 4: Necropolis
**Viability: Very Good**
- Melee shreds skeletons
- Wights dangerous but manageable
- Assassination on necromancers
- Healing items more important
- **Expected XP**: ~4,500 (combat comfortable)

#### Layer 5: Wraith Domain
**Viability: Good**
- Melee works on incorporeal?
- Need weapon with right damage type
- Fear effects disrupt combat
- High Evasion helps survival
- **Expected XP**: ~4,000 (slower but steady)

#### Layer 6: Inner Sanctum
**Viability: Good**
- Melee matches elite enemies
- Olog-hai slugfests
- Black Númenórean assassinations
- Resource intensive
- **Expected XP**: ~3,500 (tough fights)

#### Layer 7: Pits of Despair
**Viability: Moderate**
- Sauron fight is brutal melee
- May need hit-and-run tactics
- Extraction after combat
- **Expected XP**: ~2,500 (boss difficulty)

### Total Expected XP: ~29,500

### Skill Points Affordable: ~23

### Build Assessment
**Verdict: STRONG but risky**

Aragorn build gets the most kill XP but takes more damage. The stealth investment enables assassination chains that make melee safer. Strongest in Layers 2-4.

### Recommendations
- Prioritize healing items
- Consider heavy armor (if stealth allows)
- Vanish as escape option

## ARCHETYPE 4: GANDALF (Stealth Lore)

### Character Concept
- **Fantasy**: Wizard, use knowledge and song to control battlefield
- **Primary Skills**: Will, Song, Perception, Stealth
- **Playstyle**: Debuff enemies, buff self, avoid direct combat

### Build Path
```
Starting Skills:
- Will 4
- Song 3
- Perception 3
- Stealth 2

Key Abilities:
- Song of Lorien (Song 6) - enemies sleep
- Song of Staying (Song 4) - resist fear
- Channeling (Will 6)
- Fade (Stealth 8)
```

### Layer-by-Layer Playtest

#### Layer 1: Forest Breach
**Viability: Good**
- Song of Lorien puts spiders to sleep
- Staff combat when needed
- Good perception for secrets
- Slower start due to Song setup time
- **Expected XP**: ~3,500 (setup time costs efficiency)

#### Layer 2: Orc Warrens
**Viability: Good**
- Song of Lorien chains through orc groups
- Can put whole rooms to sleep
- Staff finishes sleeping targets
- Wargs resist Song?
- **Expected XP**: ~4,000 (Song effective on orcs)

#### Layer 3: Torture Halls
**Viability: Moderate**
- Ghouls immune to Song of Lorien (NO_SLEEP)
- Sorcerers contest Will checks
- Song of Silence useful here
- More challenging than earlier layers
- **Expected XP**: ~3,500 (Song less universal)

#### Layer 4: Necropolis
**Viability: Challenging**
- Undead immune to sleep Songs
- Need alternate Song strategies
- Song of Silence on necromancers
- High Will resists undead effects
- **Expected XP**: ~3,000 (Song struggles)

#### Layer 5: Wraith Domain
**Viability: Moderate**
- Song of Staying counters fear
- Wraiths immune to sleep
- Will investment pays off
- Control-based play
- **Expected XP**: ~3,000 (Will-based survival)

#### Layer 6: Inner Sanctum
**Viability: Challenging**
- Elite enemies high Will
- Song effects resisted
- May need combat backup
- Support role if grouped
- **Expected XP**: ~2,500 (Song less effective)

#### Layer 7: Pits of Despair
**Viability: Moderate**
- Sauron high Will resist
- Song of Staying crucial for morale
- Extraction Song can help allies
- Control strategy for escape
- **Expected XP**: ~2,000 (support/control)

### Total Expected XP: ~21,500

### Skill Points Affordable: ~20

### Build Assessment
**Verdict: VIABLE but niche**

Gandalf build is the weakest of the four but still playable. Song of Lorien is fantastic in Layers 1-2 but falls off against undead. Best as hybrid with another skill tree.

### Recommendations
- Pair with Archery or Melee investment
- Song of Silence for casters
- Accept combat role in undead layers

## COMPARATIVE ANALYSIS

### XP Comparison

| Archetype | Total XP | Skill Points | Viability |
|-----------|----------|--------------|-----------|
| Aragorn (Melee) | 29,500 | 23 | A |
| Faramir (Archer) | 26,500 | 22 | A |
| Bilbo (Pure Stealth) | 23,000 | 21 | B+ |
| Gandalf (Lore) | 21,500 | 20 | B |

### Layer Strength Matrix

| Archetype | L1-2 | L3-4 | L5-6 | L7 |
|-----------|------|------|------|-----|
| Bilbo | A | B+ | C+ | B |
| Faramir | A | A- | B | B+ |
| Aragorn | A | A | B+ | B- |
| Gandalf | B+ | B | B- | B |

### Key Findings

1. **All four archetypes are viable** - No build is unplayable
2. **Stealth investment pays off** - Even Aragorn benefits from Assassination
3. **Undead layers challenge pure stealth** - Layers 4-5 hardest for non-combat
4. **XP gap narrowed** - Pure stealth (21 skill points) vs combat (23 skill points) is acceptable
5. **Hybrid builds optimal** - Faramir (stealth+archery) is strongest overall

## RECOMMENDATIONS

### For Alpha 0.4.0 (Ship As-Is)
The archetype validation passes. All four builds can complete the game with appropriate challenge scaling.

### For Future Releases

1. **Song vs Undead**: Consider Song that affects undead (Song of Light?)
2. **Layer 4-5 Sleepers**: Add non-undead targets (captured prisoners? tortured humans?)
3. **Lore Build Support**: More Will-based combat options
4. **Skill Synergies**: Abilities that combine Stealth + Melee better

---

## CONCLUSION

**All four archetypes validated.** The Alpha 0.4.0 changes create a stealth-first game where:

- **Bilbo** (pure stealth) is challenging but rewarding
- **Faramir** (stealth archer) is the new optimal build
- **Aragorn** (stealth melee) remains strong
- **Gandalf** (stealth lore) is niche but playable

The XP rebalance ensures stealth builds don't fall behind. The vault redesigns provide tactical options. The monster changes create clear stealth decision points.

**Ship it.**
