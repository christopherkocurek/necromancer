## Break-Even Analysis

### Minimum Investment: Basic Smithing (Weaponsmith)

**Cost:** ~800 XP (Smithing 2 + Weaponsmith ability)

**Returns:**
- Can create weapons at forge
- Requires forge uses + materials
- No guaranteed power level over found items

**Value:** Low to Medium
- Useful for specific builds needing particular weapon types
- Not worth it if you find good weapons naturally

### Medium Investment: Full Creation (All Base Abilities)

**Cost:** ~2,000 XP (Smithing 4 + Weaponsmith + Armoursmith + Jeweller)

**Returns:**
- Create any base item type
- Auto-identify jewelry
- Customize gear for your build

**Value:** Medium
- Good for planned builds
- Provides reliable gear access

### Standard Investment: Reforge Access

**Cost:** ~2,750 XP (Smithing 5 + abilities up to Reforge)

**Returns:**
- Convert 2 Broken Glowing items → enchanted item
- XP cost: 500 per Reforge
- Recycles otherwise useless broken items

**Value:** Medium to High
- Broken Glowing items are common drops
- Enchanted items are consistently useful
- Each Reforge costs 500 XP on top of ability investment

### High Investment: Reclaim Access

**Cost:** ~4,550 XP (Smithing 7 + abilities up to Reclaim)

**Returns:**
- Convert 2 Broken Strange items → random artifact
- XP cost: artifact_level × 50 per Reclaim
- Access to artifact pool without finding them

**Value:** High
- Broken Strange items are valuable drops
- Artifacts significantly boost power
- Total cost per artifact: 250-1000 XP (level 5-20)

### Maximum Investment: Masterwork Access

**Cost:** ~5,600 XP (Smithing 8 + abilities up to Masterwork)

**Returns:**
- Convert 4 Broken Strange items → best artifact of type
- XP cost: artifact_level × 100 per Masterwork
- Guaranteed top-tier artifact

**Value:** Very High
- 4 Broken Strange is expensive but achievable
- Best artifact in category is extremely powerful
- Total cost per masterwork: 500-2000 XP (level 5-20)

## Comparative Analysis: Smithing vs Combat Skills

### Combat Build (No Smithing)

**Typical allocation for Melee build:**
- Melee 8: 3,600 XP
- Evasion 6: 2,100 XP
- Will 3: 600 XP
- Perception 2: 300 XP
- Abilities: ~3,000 XP
- **Total: ~9,600 XP**

**Gear acquisition:** Found items only (RNG dependent)

### Hybrid Build (Smithing 5)

**Allocation:**
- Melee 6: 2,100 XP
- Evasion 5: 1,500 XP
- Smithing 5: 1,500 XP + abilities ~1,250 XP
- Will 2: 300 XP
- Abilities: ~2,000 XP
- **Total: ~8,650 XP**

**Gear acquisition:** Created weapons + Reforge for enchanted items

### Smith-Focused Build (Smithing 8)

**Allocation:**
- Melee 4: 1,000 XP
- Evasion 4: 1,000 XP
- Smithing 8: 3,600 XP + abilities ~3,500 XP
- Will 2: 300 XP
- **Total: ~9,400 XP**

**Gear acquisition:** Custom artifacts via Masterwork

### Per-Point Value

| Investment | XP Cost | Benefit |
|------------|---------|---------|
| Melee +1 | 100-2000 | +10% melee hit chance |

| Masterwork use | 500-2000 | Best artifact of type |

### ROI Analysis

**Reforge ROI:**
- Cost: 500 XP + 2 Broken Glowing
- Return: Enchanted item worth ~1000 XP of combat effectiveness
- ROI: ~100% (doubling value of broken items)

**Reclaim ROI:**
- Cost: 500 XP (avg) + 2 Broken Strange
- Return: Artifact worth ~3000+ XP of combat effectiveness
- ROI: ~500% (excellent return)

**Masterwork ROI:**
- Cost: 1000 XP (avg) + 4 Broken Strange
- Return: Best artifact worth ~5000+ XP of combat effectiveness
- ROI: ~400% (still excellent)

## The Expertise Problem

### With Expertise (Level 6)

**Effect:** Negates XP and stat costs for forging

**Impact on Reforge/Reclaim/Masterwork:**
- Reforge: Free (was 500 XP)
- Reclaim: Free (was level × 50)
- Masterwork: Free (was level × 100)

**Analysis:**
If Expertise removes XP costs, then:
- Investment: ~3,600 XP to get Expertise
- After: Unlimited Reforge/Reclaim/Masterwork for free
- ROI: Infinite after ~7 uses

**Conclusion:** Expertise may be overpowered if it removes these costs.

## Opportunity Cost Analysis

### XP Spent on Smithing vs Combat

**Question:** Is 4,500 XP in Smithing better than 4,500 XP in combat skills?

**Combat 4,500 XP:**
- Melee 8 OR
- Evasion 8 OR
- Melee 6 + Evasion 4

**Smithing 4,500 XP:**
- Smithing 7 with Reclaim ability
- Can create artifacts from broken items
- Each artifact worth ~3000 XP of combat effectiveness

**Break-even point:**
- If you can create 2 artifacts via Reclaim, you've matched combat investment
- If you can create more, Smithing wins

## Recommendations

### For Balance

1. **Keep Reforge at 500 XP** - Fair cost for enchanted items
2. **Keep Reclaim scaling** - level × 50 is reasonable
3. **Adjust Masterwork** - level × 100 may be too cheap for guaranteed best
4. **Nerf Expertise** - Should reduce costs, not eliminate them

### Proposed Expertise Change

**Current:** Negates all XP and stat costs
**Proposed:** Reduces XP and stat costs by 50%

**Impact:**
- Reforge: 250 XP (was free)
- Reclaim: level × 25 (was free)
- Masterwork: level × 50 (was free)
- Still valuable, but maintains XP economy

## Summary

### Current State

- Smithing 5-8 is the sweet spot for value
- Smithing 10 (Grace) provides little incentive
- Smithing 11-20 provides no new abilities
- Expertise may break XP economy

### Value Proposition Score

| Investment Level | Value Score | Recommendation |
|------------------|-------------|----------------|
| Smithing 0 (none) | 0/10 | Default for non-smiths |
| Smithing 5 (Reforge) | 7/10 | Good hybrid option |
| Smithing 7 (Reclaim) | 8/10 | Strong for artifact hunters |
| Smithing 8 (Masterwork) | 9/10 | Excellent for dedicated smiths |
| Smithing 10 (Grace) | 6/10 | Diminishing returns |
| Smithing 11-20 | 3/10 | Little benefit, not recommended |

### Key Finding

**The "Smithing Cliff" at level 10 is real.** There is no compelling reason to invest in Smithing above level 10 unless:
1. Attempting extremely high-difficulty items
2. Compensating for poor forge quality
3. New high-level abilities are added
