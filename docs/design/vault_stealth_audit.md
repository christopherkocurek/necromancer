# Vault Stealth Audit - The Necromancer
## VAULT-003: Stealth Gameplay Evaluation

---

## Scoring Criteria (0-5 Scale)

Each vault evaluated on 5 factors (+1 point each):
1. **Multiple Entry Points** - Can approach from different directions?
2. **Cover/Pillars** - Line-of-sight breaks for hiding?
3. **Patrol Routes Possible** - Space for patrol behavior?
4. **Chokepoints** - Bottlenecks that enable ambush?
5. **Line-of-Sight Breaks** - Can avoid detection?

**Score Interpretation:**
- 0-1: Combat-focused, avoid stealth rework
- 2-3: Adequate stealth potential
- 4-5: Excellent stealth design

## DESIGN RECOMMENDATIONS

### Add to Priority Vaults:
1. **Multiple entry points** - Side corridors, secret doors
2. **Pillar/cover placement** - Breaking line of sight
3. **Patrol loop paths** - Space for monster movement
4. **Alternative routes** - Stealth bypass options
5. **Darkness zones** - Shadow areas for hiding

### Vault Archetypes Needed:
1. **"The Bypass"** - Secret passage around danger
2. **"The Patrol Loop"** - Monsters on timed route
3. **"The Sleep Chamber"** - Sleeping enemies to sneak past
4. **"The Lure"** - Distraction mechanic possible
5. **"The Vertical"** - Multi-level approach options

### Quick Fixes:
- Add secret doors (s) to existing vaults
- Add rubble (:) for cover
- Widen corridors for patrol space
- Add alternative entrances ($)
