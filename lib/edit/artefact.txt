# Artifact definitions for THE NECROMANCER
# A SIL-Q fork set in Third Age Dol Guldur

# === Understanding artefact.txt ===
#
# N: serial number : item name
# G: char : attr
# I: tval : sval : pval
# B: skilltype / abilitynum : ...
# W: depth : rarity : weight : cost
# P: attack bonus : damage dice : evasion bonus : protection dice
# F: flag | flag | etc
# D: text
#
# === Important Notes ===
#
# (1) Indices 1-19: Special artifacts (rings, amulets, lights, crowns)
# (2) Indices 20-139: Normal artifacts (armor, weapons)
# (3) Indices 175-179: Quest items (Ring of Thráin, Key to Erebor)
# (4) Indices 182-198: Smithing templates (DO NOT MODIFY)
#
